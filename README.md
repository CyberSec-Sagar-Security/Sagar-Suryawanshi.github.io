## âœï¸ Customization

### Update Personal Information

Edit `js/main.js` to update:
- Skills data
- Certifications
- Projects
- Experience
- Education
- Achievements

### Modify Theme Colors

Edit CSS variables in `css/style.css`:
```css
:root {
    --accent-primary: #06b6d4;    /* Main accent color */
    --accent-secondary: #22d3ee;  /* Secondary accent */
    --bg-primary: #0a0f1c;        /* Background color */
}
```

### Add Resume PDF

Place your resume PDF in the `assets/` folder and name it `Sagar_Suryawanshi_Resume.pdf` or update the link in `index.html`.

### Update Contact Information

Edit the contact links in:
- Hero section social links
- Contact section
- Footer
